# Santa’s Perfect Christmas Melon
Santa is at the North Pole Christmas Market looking for the **perfect Christmas melon** to take on his holiday journey. All melons are lined up in order, each with a known weight.

Santa wants to choose the melon whose weight is closest to the **average weight of all melons** — not too heavy, not too light, but just right for Christmas magic!
---
### Your Task

Write a program that:

1. Calculates the **average weight** of all the Christmas melons.
2. Finds the **index (number)** of the melon whose weight is **closest to this average**.
3. Prints the melon number and the average weight (rounded to two decimal places).

---

#### Input

First value: **n** — the number of melons.
Next values: **n real numbers** — the weights of each melon.

---

#### Output

Print on one line:

`melonNumber averageWeight`

---

### Example

Input:

```bash
4
4 3 7 8
```

Output:

```bash
1 4.00
```
